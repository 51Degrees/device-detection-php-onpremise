fiftyone\pipeline\core\PipelineBuilder Class Reference

Detailed Description

A PipelineBuilder generates a Pipeline object Before construction of the Pipeline, FlowElements are added to it There are also options for how JavaScript is output from the Pipeline.

Examples
onpremise/failureToMatch.php, onpremise/gettingStartedConsole.php, onpremise/gettingStartedWeb.php, onpremise/manualDataUpdate.php, onpremise/matchMetrics.php, onpremise/metadataConsole.php, and onpremise/userAgentClientHints-Web.php.
